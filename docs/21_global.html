Summary. Global parameters taken from Schmitt et al. (2024).

We first used back parameters calibrated for Tapajos from Schmitt et al. (2024) for a first test:

rcontroll::generate_parameters(
  CR_a = 2.45,
  CR_b = 0.7565,
  m = 0.040,
  m1 = 0.040,
  pheno_a0 = 0.2,
  pheno_b0 = 0.015,
  pheno_delta = 0.2

  filter(!(param %in% c(
    "nbiter", "HEIGHT", "length_dcell", "NV", "NH", "nbout", "nbspp",

  ))) %>%
